Rinne Sharingan

Le Rinne Sharingan est une forme combinée du Rinnegan et du Sharingan, considéré comme l'ancêtre de ces deux pouvoirs. Il apparaît sous la forme d'un œil avec un motif de tomoe entourant un anneau, et il est capable de lancer l'ultime illusion, le Tsukuyomi Infini. Cette technique emprisonne tous les êtres vivants dans un monde de rêves éternels, les plongeant dans une illusion collective de bonheur tandis que leur énergie est drainée pour nourrir l'utilisateur. Le Rinne Sharingan permet également de se déplacer entre différentes dimensions.

Utilisateurs principaux:

Kaguya Ōtsutsuki : Kaguya, la première à posséder le Rinne Sharingan, utilise ce pouvoir pour tenter de soumettre l'humanité entière à son contrôle via le Tsukuyomi Infini.
Madara Uchiha : Dans sa forme finale, Madara acquiert le Rinne Sharingan, qu'il utilise pour déployer le Tsukuyomi Infini dans une tentative de réaliser ses objectifs de domination mondiale.
